verified_user User Security

User accounts are sent securely via HTTPS POST requests to the server, and stored in a MongoDB database. Some data, such as word-lists, are stored publicly on repl.it.
Usernames and some data is stored on MongoDB, but passwords are hashed and salted using bcrypt before storing to the database. Sessions expire after sixty minutes (one minute for super-administrators) and all tokens are validated server-side.
Administrators cannot access user data, they may only modify user account passwords and create new accounts. Please contact BetaOS to request account recovery, deletion or modification.
We also log a user-token every time a user logs in, generated securely using window.crypto.
We take user security very seriously and will work to fix all bugs as soon as possible. Please submit a bug report if you find any vulnerabilities or would like more features!

build About BetaUtilities

BetaUtilities is a bot that connects to the online discussion site Euphoria's "rooms". It automatically responds to commands such as !help and provides a variety of features:
(The bot sometimes appears as a different username according to the room it is currently in.)
!ping/!ping @BetaUtilities - Checks if BetaUtilities is online.
!help @BetaUtilities/!contact @BetaUtilities - Displays a telephone-button style menu for accessing almost all features in the bot. Enter numbers, the english form of the letter or the emoji :short-code: for the choice you would like to make to interact with the menu. Call ends after around 30 seconds.
!kill @BetaUtilities - Disconnects BetaUtilities from the room.
!pause @BetaUtilities - Deactivates BetaUtilities without disconnecting it from the current room. BetaUtilities will no longer respond to your commands until you run:
!restore @BetaUtilities - Re-activates BetaUtilities.
!uptime @BetaUtilities - Displays how long the current instance has been up.
!wordle - Plays wordle with you! BetaUtilities comes with a built-in word-list so you can't just enter sequences of five letters.
!leetlen't - Plays a BetaOS version of Leetle with you - it works the same as Wordle but you can enter any sequences of 5 alphanumeric characters.
!unblock [link] - Sends your link through a proxy to un-block it for school networks.
!src @BetaUtilities - Sends you a link to the source-code.
!runStats @BetaUtilities - Shows you statistics on how many times BetaUtilities has been run, and how many times it has been @-mentionned.
!creatorInfo,!version - Pretty self-explanatory.
!status @BetaUtilities - Sends this site to the room.
!activeRooms @BetaUtilities - Shows all the rooms BetaUtilities is active in.
!mitoseTo &[ROOM] as @[NAME] - Sends BetaUtilities to a new room as another username.
!rating @BetaUtilities - Accesses the current call satisfaction rating for BetaUtilities. You can rate us by using the help-menu.
!work/!work @user/!sleep/!sleep @user - Screams at you or any user to GO TO SLEEP or GET TO WORK.
!wake/!wake @user/!play/!play @user - Stops screaming at you or any user.
!antispam - Configures AntiSpam activation. Requires a code entry.
Some features are hidden! ask BetaOS about them, or find us in &xkcd to ask about the features!

speed AntiSpam Feature

BetaUtilities detects automatically when it has been triggered too many times in 5 minutes (usually 5 messages) and alerts the user. It is automatically paused after 10 messages have been sent in 5 minutes, unless the !antispam command has been used and AntiSpam has been disabled. This feature is both in BetaUtilities on Euphoria and BetaOS Systems ONLINE.

hub BetaOS Systems ONLINE

Sign-up and login are pretty self-explanatory. Visit the config page to modify your password, or as an administrator, manage other accounts. Super-administrators can run code using the commands box.
The Support system works the same as BetaUtilities on Euphoria.
Enter !renick @[NEWUSERNAME] to rename yourself. The SysLog can only be accessed by administrators, and the status-page alerts you when BetaUtilities is down.
